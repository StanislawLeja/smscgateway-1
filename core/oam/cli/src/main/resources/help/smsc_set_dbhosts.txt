Name
	smsc set dbhosts <ip-address>

SYNOPSIS
	smsc set dbhosts

DESCRIPTION
	This command is used to set the host-ip address for Cassandra Database
	access.

EXAMPLES
	smsc set dbhosts 127.0.0.1
