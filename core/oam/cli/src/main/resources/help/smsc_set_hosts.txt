Name
	smsc set hosts

SYNOPSIS
	smsc set hosts <host-ip:port>

DESCRIPTION
	This command is used to set the host-ip address and port for Cassandra Database
	access.

EXAMPLES
	smsc set hosts 127.0.0.1:9160
