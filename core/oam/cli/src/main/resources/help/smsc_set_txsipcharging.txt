Name
	smsc set txsipcharging

SYNOPSIS
	smsc set txsipcharging <None | Selected | All>

DESCRIPTION
	This command is used to set the value of the parameter 'txsipcharging' 
	is set to 'none', 'selected' or 'all'. This option defines if SIP originated 
	messages will be charged by OCS via Diameter before delivery.
